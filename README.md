# LangChain Chatbot 🤖

An intelligent chatbot application built using LangChain, Gemini API, and Python.

## 🚀 Features
- AI-powered chatbot
- Uses LangChain for chaining prompts
- Integrated with Gemini API
- Simple and interactive UI

## 🛠️ Tech Stack
- Python
- LangChain
- Node.js
- Vite

## 📦 Installation

```bash
npm install
```

## ▶️ Run the Project

```bash
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file and add:

```
GEMINI_API_KEY=your_api_key_here
```
